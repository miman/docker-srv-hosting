# Netbird

Netbird](https://netbird.io/) is a service that lets you build a Wireguard-based VPN for yourself. You can connect to all your devices securely over the internet.

This docker-compose.yml file installs the client side of [Netbird](https://netbird.io/).

For this to work you either need to use the cloud based controlplane from Netbird or install your own local controlplane on your own server. 

See the [server/readme.md](server/readme.md) file for more information.
