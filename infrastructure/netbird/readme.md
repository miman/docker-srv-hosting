# Netbird

[Netbird](https://netbird.io/) is a service that lets you install your own secure wireguard-based VPN network.

This is similar to Tailscale, but open source.

It also operates from EU, which means it conforms to the EU's strong data privacy laws (GDPR, etc).

It is a great alternative to commercial VPNs like NordVPN, ProtonVPN, etc., but with the added benefit of being open source.

You can either use the cloud based controlplane (easier) or install your own local controlplane on your own server (full privacy).
This project provides instructions on how to do both.
