# Traccar - GPS Tracking Platform

[Traccar](https://www.traccar.org/) is a free and open-source GPS tracking platform. It is used to track the location of vehicles, people, and other assets.

It is a web application that allows you to view the location of your devices on a map, and to receive notifications when they move or stop.

## Installation & Running

To deploy Traccar, you can use the provided installation script:

```bash
chmod +x install.sh
./install.sh
```
