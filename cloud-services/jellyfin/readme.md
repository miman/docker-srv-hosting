# Jellyfin

This folder contains docker-compose files used to host [Jellyfin](https://jellyfin.org/) on Docker or Podman.

[Jellyfin](https://jellyfin.org/) is a free and open-source media system for streaming media.

## Synology

Under the folder **{prj-root}synology-nas/jellyfin** there is a script tuned for use on a Synology NAS.
