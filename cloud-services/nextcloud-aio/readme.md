# Nextcloud AIO

[Nextcloud AIO](https://nextcloud.com/aio/) is a open source software for file hosting and collaboration.

In contrast to [Nextcloud](../nextcloud/readme.md) this is a complete all-in-one solution that includes:

- Nextcloud
- Database
- Reverse proxy
- Let's Encrypt for SSL certificates
- Monitoring
- Backup and restore
- And much more

Think of it like a Google Docs or Microsoft Ofice 365 running locally. 

It also include other useful features like chat, video calls, and more.
