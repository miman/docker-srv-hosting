# Local cloud recommended setup

This is what I would recommend if you want to replace most of the commercial cloud services with local alternatives (like your local Google Drive or Office 365)

# Recommended services
| Service             | Description                                            |
| ------------------- | ------------------------------------------------------ |
| [Immich](../../cloud-services/immich/readme.md)              | Self-hosted photo & video backup (like Google/Apple Photos)                       |
| [Nextcloud](../../cloud-services/nextcloud/readme.md)           | File sync and collaboration suite (like Google Drive or Office 365)                      |
| [Vaultwarden](../../cloud-services/vaultwarden/readme.md)         | Self-hosted password manager                           |

## Optional services
These are nice services, but not everyone has these specific needs

| Service             | Description                                            |
| ------------------- | ------------------------------------------------------ |
| [Synapse](../../cloud-services/synapse/readme.md) | Matrix homeserver for secure communication (chat/video), replacement for WhatsApp, iMessage...  |
| [Home Assistant](../../cloud-services/home-assistant/ReadMe.md)      | Home automation platform                               |
| [Docmost](../../cloud-services/docmost/readme.md)             | Self-hosted documentation/wiki platform                |
| [Traccar](../../cloud-services/traccar/readme.md)             | GPS tracking platform                                  |
| [Nextcloud AIO](../../cloud-services/nextcloud-aio/readme.md)       | Easy-to-deploy Nextcloud All-in-One version, if you didn't want the default version            |


## Prerequisites

You MUST have done the [infrastructure-setup](infrastructure-setup.md) first.
