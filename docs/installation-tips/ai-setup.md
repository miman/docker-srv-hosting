# AI setup

This page describes the services that you should install to run local AI services.

## Local AI (LLM inference engine)

These services are all about large language models (LLMs), think local ChatGPT/Gemini/Claude...)

| Service             | Description                                            |
| ------------------- | ------------------------------------------------------ |
| [Ollama LLM server](../../ai/ollama/ReadMe.md)   | Local LLM (AI) server                                  |
| [Open WebUI for Ollama](../../ai/open-webui/ReadMe.md)   | Web interface for Ollama LLM                           |
| [SearXNG](../../ai/searxng/readme.md)   | Privacy-respecting metasearch engine (used for web search in local AI)                  |

## AI Images, Video, & Audio

These services allow you to run local AI Image, Video, and Audio generation.

| Service             | Description                                            |
| ------------------- | ------------------------------------------------------ |
| [ComfyUI](../../ai/comfy_ui/ReadMe.md) | Powerful and modular stable diffusion GUI/backend |
| [Lyra](../../ai/3d/Lyra/readme.md) | NVIDIA Lyra-2.0 for high-performance AI generation |

## Prerequisites

You MUST have done the [infrastructure-setup](infrastructure-setup.md) first.
